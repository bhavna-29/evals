    Prompt¶

    Manage versioned prompt templates stored in TestOps.

    Use get() to fetch a prompt by name, create() to publish a new version, and list() to browse stored prompts. Call .compile() on the object returned by get() to render the final string.

    Functions¶

    get(name, *, version=None, label=None, type='text', cache_ttl_seconds=None, fallback=None, max_retries=None, fetch_timeout_seconds=None) classmethod ¶

    Fetch a prompt and return it as a compilable PromptClient.

    Retrieves the prompt from the BrowserStack API, injects id and modelParams metadata onto the returned object, and populates the local prompt cache.

    Can be called on an AISDK instance (sdk.prompt.get(...)) or as a standalone classmethod (Prompt.get(...)). When neither version nor label is provided the API resolves the 'latest' label.

    Parameters:

    Name Type Description Default
    name str

    Name of the prompt to retrieve. Must be a non-empty string.

    		 required
    version Optional[int]

    Specific version number to fetch. Defaults to None (resolved via label).

    		 None
    label Optional[str]

    Label to fetch (e.g. 'production'). Defaults to None (falls back to 'latest').

    		 None
    type Literal['chat', 'text']

    Expected prompt type — 'text' or 'chat'. Defaults to 'text'.

    		 'text'
    cache_ttl_seconds Optional[int]

    Local in-memory cache TTL in seconds. Defaults to None (no cache TTL override).

    		 None
    fallback Union[Optional[List[ChatMessageDict]], Optional[str]]

    Fallback content used when the fetch fails. Defaults to None.

    		 None
    max_retries Optional[int]

    Maximum retry attempts on transient failures. Defaults to None (uses SDK default).

    		 None
    fetch_timeout_seconds Optional[int]

    Per-request timeout in seconds. Defaults to None.

    		 None

    Returns:

    Type Description
    PromptClient

    A PromptClient with the prompt data. Call prompt.compile(variables={...}) to render the final string.

    Raises:

    Type Description
    ValidationError

    If name is empty or invalid.
    AuthenticationError

    If the API key is invalid or expired.
    TestOpsAPIError

    If the API request fails.
    Example 
    # Via AISDK instance
sdk = AISDK(public_key="pk", secret_key="sk")
prompt = sdk.prompt.get("summarise-text", label="production")
text = prompt.compile(variables={"text": "Long article..."})

# Standalone (credentials from environment variables)
from browserstack_ai_sdk import Prompt
prompt = Prompt.get("summarise-text", label="production")

    create(*, name, prompt, type='text', labels=None, tags=None, config=None, model_params=None, commit_message=None, **kwargs) classmethod ¶

    Create a new prompt version via the BrowserStack public API.

    Supports both 'text' and 'chat' prompt types. For chat prompts, messages are validated and normalised to the server schema. Can be called on an AISDK instance (sdk.prompt.create(...)) or as a standalone classmethod (Prompt.create(...)).

    Parameters:

    Name Type Description Default
    name str

    Unique name for the prompt. Must be a non-empty string.

    		 required
    prompt Union[str, List[Dict[str, Any]]]

    For type='text': a plain Mustache string template. For type='chat': a list of message dicts, each with at least role and content keys.

    		 required
    type Optional[Literal['chat', 'text']]

    Prompt type — 'text' or 'chat'. Defaults to 'text'.

    		 'text'
    labels Optional[List[str]]

    Labels to attach to this version (e.g. ['production']). Defaults to None (no labels).

    		 None
    tags Optional[List[str]]

    Tags for categorisation. Defaults to None.

    		 None
    config Optional[Dict[str, Any]]

    Arbitrary config dict stored alongside the prompt. Defaults to None.

    		 None
    model_params Optional[ModelParams]

    Model configuration overrides. Accepts a ModelParams dict with:

    • provider (str, required) — LLM provider, e.g. "openai".
    • model (str, required) — Model name, e.g. "gpt-4o".
    • adapter (str, required) — Adapter identifier.
    • temperature (float, optional) — Sampling temperature (0.0–2.0).
    • max_tokens (int, optional) — Max tokens in the response.
    • top_p (float, optional) — Nucleus sampling probability.
    • top_k (int, optional) — Top-k sampling filter.
    • stop_sequences (list[str], optional) — Stop token list.
    • frequency_penalty (float, optional) — Penalise repeated tokens.
    • presence_penalty (float, optional) — Penalise new topics.
    • reasoning_enabled (bool, optional) — Enable chain-of-thought reasoning.
    • use_cache (bool, optional) — Enable prompt caching.

    Defaults to None.

    		 None
    commit_message Optional[str]

    Human-readable description of this version. Defaults to None.

    		 None
    **kwargs

    Additional keyword arguments. isActive=True is handled for backward compatibility by injecting 'production' into labels.

    		 {}

    Returns:

    Type Description
    PromptResponse

    A dict containing the created prompt's id, name, version, and related metadata.

    Raises:

    Type Description
    ValidationError

    If name is empty or the prompt format is incorrect for the given type.
    AuthenticationError

    If the API key is invalid or expired.
    TestOpsAPIError

    If the API request fails.
    Example 
    # Via AISDK instance
sdk = AISDK(public_key="pk", secret_key="sk")
result = sdk.prompt.create(
    name="summarise-text",
    prompt="Summarise the following: {{text}}",
    type="text",
    labels=["production"],
)
print(result["id"])

# Standalone
from browserstack_ai_sdk import Prompt
result = Prompt.create(name="my-prompt", prompt="Hello {{name}}")

    list(*, name=None, label=None, tag=None, page=1, limit=50, from_updated_at=None, to_updated_at=None, request_options=None) classmethod ¶

    		 None

    Filter by tag. Defaults to None.

    		 None

    1-based page number. Defaults to 1.

    		 1
    limit Optional[int]

    Number of prompts per page. Defaults to 50.

    		 50
    from_updated_at Optional[datetime]

    Return prompts updated at or after this UTC datetime. Defaults to None.

    		 None
    to_updated_at Optional[datetime]

    Return prompts updated at or before this UTC datetime. Defaults to None.

    		 None
    request_options

    Optional Langfuse request options (timeout, etc.). Defaults to None.

    		 None

    Returns:

    Type Description
    List[PromptResponse]

    A list of dicts, each representing a prompt version record with name, version, labels, and related metadata.

    Raises:

    Type Description
    AuthenticationError

    If the API key is invalid or expired.
    TestOpsAPIError

    If the API request fails.
    Example 
    # Via AISDK instance

# Standalone
from browserstack_ai_sdk import Prompt
prompts = Prompt.list(tag="qa", page=1)

    compile()¶

    Render a fetched prompt template with your variables. Called on the PromptClient object returned by get() — not on Prompt directly.

    Prompt templates use Mustache syntax: {{variable_name}}.

    Parameters

    Name Type Required Default Description
    variables dict No {} Key-value pairs substituted into the template placeholders.

    Returns

    • For type='text' prompts — a rendered str with all {{variable}} placeholders replaced.
    • For type='chat' prompts — a list of message dicts with variables substituted.

    Raises

    • KeyError — If a placeholder in the template has no matching key in variables.

    Example

    from browserstack_ai_sdk import AISDK

sdk = AISDK(public_key="pk", secret_key="sk")
prompt = sdk.prompt.get("my-prompt", label="production")
result = prompt.compile(variables={"topic": "AI testing"})
# result is a ready-to-use str (text prompt) or list of message dicts (chat prompt)
