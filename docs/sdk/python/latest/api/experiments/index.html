    Experiments¶

    Organise related evaluation runs into named experiments.

    An experiment ties together a dataset, a prompt, and an evaluator list so that results from multiple runs can be tracked and compared in one place. Each call to create() returns an experiment that ExperimentRuns can then execute against.

    Access via sdk.experiments or standalone via Evaluate.experiment.

    Example 
    import os
from browserstack_ai_sdk import AISDK

sdk = AISDK(
    public_key=os.environ["AISDK_PUBLIC_KEY"],
    secret_key=os.environ["AISDK_SECRET_KEY"],
)
exp = sdk.experiments.create({
    "name": "answer-quality-gpt4o",
    "evaluatorListId": "evl_a1b2c3d4",
    "datasetRunTagId": "tag_x9y8z7",
})
print(exp["id"])   # "exp_f1e2d3c4"

    Functions¶

    create ¶

    create(request_data)

    Create a new experiment on the BrowserStack Evals platform.

    Accepts either a tag-based request (links to a dataset run via a tag) or a prompt-based request (links directly to a prompt and dataset).

    Parameters:

    Name Type Description Default
    request_data Union[CreateExperimentWithTagRequest, CreateExperimentWithPromptRequest]

    A dict with required keys:

    • name (str): Unique experiment name.
    • evaluatorListId (str): ID of the evaluator list to use.
    • datasetRunTagId (str) or both promptId (str) and datasetId (str).
    		 required

    Returns:

    Type Description
    ExperimentResponse

    A dict containing the created experiment's metadata, including its id.

    Raises:

    Type Description
    ValidationError

    If required fields are missing or mutually exclusive fields are both present.
    AuthenticationError

    If the API key is invalid or expired.
    AuthorizationError

    If the caller lacks permission.
    TestOpsAPIError

    If the API request fails for any other reason.
    Example 
    import os
from browserstack_ai_sdk import AISDK

# Via AISDK instance — tag-based experiment
sdk = AISDK(
    public_key=os.environ["AISDK_PUBLIC_KEY"],
    secret_key=os.environ["AISDK_SECRET_KEY"],
)
exp = sdk.experiments.create({
    "name": "answer-quality-gpt4o",
    "evaluatorListId": "evl_a1b2c3d4",
    "datasetRunTagId": "tag_x9y8z7",
})
print(exp["id"])     # "exp_f1e2d3c4"
print(exp["status"]) # "PENDING"

    import os
from browserstack_ai_sdk import Evaluate

# Standalone — prompt-based experiment
# Set AISDK_PUBLIC_KEY and AISDK_SECRET_KEY in your environment first
exp = Evaluate.experiment.create({
    "name": "faithfulness-claude-sonnet",
    "evaluatorListId": "evl_a1b2c3d4",
    "promptId": "pmt_z7y8x9",
    "datasetId": "ds_m3n4o5",
})
print(exp["id"])

    list ¶

    list(limit=50, page=1)

    List experiments with pagination.

    Parameters:

    Name Type Description Default
    limit int

    Maximum number of experiments to return per page. Must be between 1 and 100. Defaults to 50.

    		 50
    page int

    1-based page number. Defaults to 1.

    		 1

    Returns:

    Type Description
    ListExperimentsResponse

    A dict with experiments (list of experiments) and totalCount (total count) keys.

    Raises:

    Type Description
    ValidationError

    If limit or page are out of range.
    AuthenticationError

    If the API key is invalid or expired.
    AuthorizationError

    If the caller lacks permission.
    TestOpsAPIError

    If the API request fails.
    Example 
    import os
from browserstack_ai_sdk import AISDK

# Via AISDK instance
sdk = AISDK(
    public_key=os.environ["AISDK_PUBLIC_KEY"],
    secret_key=os.environ["AISDK_SECRET_KEY"],
)
result = sdk.experiments.list(limit=10, page=1)
print(f"Total experiments: {result['totalCount']}")
for exp in result["experiments"]:
    print(exp["name"], "—", exp["id"])
# answer-quality-gpt4o — exp_f1e2d3c4
# faithfulness-claude-sonnet — exp_a5b6c7d8

    import os
from browserstack_ai_sdk import Evaluate

# Standalone
result = Evaluate.experiment.list(limit=20)
for exp in result["experiments"]:
    print(exp["name"])

    get ¶

    get(experiment_id)

    Retrieve a single experiment by its ID.

    Parameters:

    Name Type Description Default
    experiment_id str

    The unique ID of the experiment to retrieve.

    		 required

    Returns:

    Type Description
    ExperimentResponse

    A dict containing the experiment's metadata.

    Raises:

    Type Description
    ValidationError

    If experiment_id is empty.
    ExperimentNotFoundError

    If no experiment with the given ID exists.
    AuthenticationError

    If the API key is invalid or expired.
    AuthorizationError

    If the caller lacks permission.
    TestOpsAPIError

    If the API request fails for any other reason.
    Example 
    import os
from browserstack_ai_sdk import AISDK

# Via AISDK instance
sdk = AISDK(
    public_key=os.environ["AISDK_PUBLIC_KEY"],
    secret_key=os.environ["AISDK_SECRET_KEY"],
)
exp = sdk.experiments.get("exp_f1e2d3c4")
print(exp["name"])   # "answer-quality-gpt4o"
print(exp["status"]) # "COMPLETED"

    import os
from browserstack_ai_sdk import Evaluate

# Standalone
exp = Evaluate.experiment.get("exp_f1e2d3c4")
print(exp["evaluatorListId"])
