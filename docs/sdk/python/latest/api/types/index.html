    Types Reference¶

    Type definitions for TestOps experiment runs.

    This module provides TypedDict classes and other type definitions for better type safety when working with experiment runs.

    ExperimentRunStatus ¶

    Constants for experiment run statuses.

    Use these values when filtering experiment runs by status or when checking the status field on a returned ExperimentRunResponse.

    Example 
    from browserstack_ai_sdk.types import ExperimentRunStatus

run = sdk.experiment_runs.find("run-abc")
if run["status"] == ExperimentRunStatus.COMPLETED:
    print("Run finished successfully.")

    Attributes¶

    PENDING ¶
    PENDING = 'PENDING'

    Run has been created but not yet started.

    RUNNING ¶
    RUNNING = 'RUNNING'

    Run is currently executing evaluations.

    COMPLETED ¶
    COMPLETED = 'COMPLETED'

    Run has finished successfully.

    FAILED ¶
    FAILED = 'FAILED'

    Run encountered an error and did not complete.

    CANCELLED ¶
    CANCELLED = 'CANCELLED'

    Run was cancelled before completion.
